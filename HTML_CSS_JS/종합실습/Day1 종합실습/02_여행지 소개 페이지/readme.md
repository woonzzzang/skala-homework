📚 수행 내용  
- 시맨틱 태그와 미디어 요소를 사용해 여행지 소개 페이지를 만들고, 고정 내비게이션과 앵커 이동, 이미지 갤러리, 영상, 여행 팁 영역을 구성했습니다.

## Doing

이번 과제에서는 여행지 소개 페이지를 만들면서 HTML 시맨틱 요소와 미디어 요소를 함께 사용해봤습니다. 단순히 화면에 내용을 나열하는 것이 아니라, `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`가 각각 어떤 역할을 가지는지 생각하면서 페이지 구조를 나누는 데 중점을 두었습니다.

페이지 상단에는 여행지의 분위기를 보여줄 수 있는 히어로 영역을 만들었습니다. 배경에는 이미지를 넣고 그 위에 `gradient` 오버레이를 겹쳐 텍스트가 잘 보이도록 했습니다. 히어로 안의 문구는 `position: absolute`와 `transform`을 사용해 가운데에 배치했습니다.

내비게이션은 `position: fixed`와 `z-index`를 적용해 화면을 아래로 내려도 상단에 계속 고정되도록 만들었습니다. 각 메뉴는 `href="#id"` 형식으로 섹션 id와 연결했고, `html { scroll-behavior: smooth; }`를 적용해 메뉴를 클릭했을 때 해당 위치로 부드럽게 이동하도록 했습니다. 고정 내비게이션이 섹션 제목을 가리지 않도록 각 섹션에는 `scroll-margin-top`도 함께 적용했습니다.

명소 갤러리 영역은 `figure`와 `figcaption`을 사용해 이미지와 설명을 하나의 묶음으로 구성했습니다. 도쿄 디즈니씨, 오사카 글리코상, 후쿠오카 이치란 라멘 본점 사진을 각각 카드 형태로 배치했고, 모든 이미지에는 의미 있는 `alt`를 작성했습니다. 이미지 카드에는 `:hover`와 `transform: scale()`을 적용해 마우스를 올렸을 때 사진이 살짝 확대되도록 만들었습니다.

먹거리 섹션은 본문 역할의 `article`과 보조 정보 역할의 `aside`를 나누어 구성했습니다. `grid-template-columns`를 사용해 본문과 팁 영역이 2단으로 보이도록 배치했고, aside에는 여행 중 참고하면 좋은 간단한 팁을 넣었습니다.

영상 섹션에는 `video` 요소를 사용했습니다. `controls`를 적용해 사용자가 직접 재생과 정지를 조작할 수 있도록 했고, `poster`를 넣어 영상이 재생되기 전에도 화면이 비어 보이지 않도록 했습니다. `source` 태그를 사용해 실제 영상 파일을 연결했습니다.

마지막으로 footer에는 페이지 최상단으로 돌아갈 수 있는 `↑ 맨 위로` 링크를 넣었습니다. 이 링크도 `#top` 앵커와 연결되어 있어 페이지 내부 이동을 한 번 더 확인할 수 있도록 구성했습니다.

## 📝 자기 평가

이번 과제에서는 시맨틱 태그를 사용하는 이유를 더 분명하게 이해할 수 있었습니다. 화면상으로는 `div`만 사용해도 비슷하게 만들 수 있지만, `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`처럼 역할이 있는 태그를 사용하니 코드 구조가 훨씬 읽기 쉬워졌습니다. 특히 `main`은 페이지의 핵심 내용을 담는 영역이고, `aside`는 본문을 보조하는 정보를 담는 영역이라는 점을 직접 배치해보며 익힐 수 있었습니다.

이미지와 설명을 따로 두는 대신 `figure`와 `figcaption`으로 묶어보면서, 미디어 자료에도 구조가 필요하다는 것을 알게 되었습니다. 또 `img`의 `alt`는 단순히 형식상 넣는 속성이 아니라 이미지가 보이지 않거나 스크린 리더를 사용할 때 내용을 전달하는 역할을 한다는 점을 다시 확인했습니다.

`video` 요소에서는 `controls`, `poster`, `source`의 역할을 구분해서 사용할 수 있었습니다. `controls`가 있어야 사용자가 영상을 직접 재생하거나 멈출 수 있고, `poster`를 넣으면 영상이 시작되기 전에도 빈 화면이 아니라 대표 이미지가 보인다는 점이 실용적이었습니다.

CSS에서는 `position: fixed`로 내비게이션을 고정하고, `z-index`로 다른 요소보다 위에 보이게 만드는 과정을 연습했습니다. 다만 고정된 내비게이션은 섹션 제목을 가릴 수 있어서 `scroll-margin-top`처럼 앵커 이동 시 위치를 보정하는 속성도 함께 고려해야 한다는 것을 배웠습니다.

히어로 영역에서는 이미지 위에 `gradient`를 겹쳐 텍스트 가독성을 높였고, `position: absolute`와 `transform`으로 중앙 정렬을 적용했습니다. 카드에는 `:hover`와 `scale`을 사용해 마우스를 올렸을 때 이미지가 살짝 확대되도록 만들었습니다. 작은 효과지만 사용자가 페이지와 상호작용하고 있다는 느낌을 줄 수 있다는 점이 좋았습니다.

이번 실습을 하면서 HTML 구조, 미디어 요소, 앵커 이동, 고정 내비게이션, hover 효과가 서로 연결되어 하나의 페이지 경험을 만든다는 것을 배웠습니다.
