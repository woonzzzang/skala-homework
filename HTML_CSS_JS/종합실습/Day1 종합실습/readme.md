# Day1 종합실습

🗓️ 수행 날짜 : 2026-07-15 ~ 2026-07-18  
👤 작성자 : 4기 광주 3반 정다운  

HTML과 CSS의 기본 개념을 익힌 뒤, 작은 실습에서 종합 페이지 제작까지 단계적으로 진행한 Day1 실습 모음입니다. 처음에는 HTML 문서 구조와 링크 사용법을 확인했고, 이후에는 회원가입 폼과 여행지 소개 페이지를 만들며 form, semantic tag, media element, CSS 배치와 상태 스타일을 함께 적용했습니다.

## 00_Day1 실습

Day1에서 배운 HTML 기초를 작은 단위로 확인한 실습입니다.

### 01_web_workflow

웹 페이지가 브라우저에 표시되는 기본 흐름을 확인했습니다. HTML의 `head`와 `body` 구조를 구분하고, `index.html`의 본문 문구를 수정한 뒤 Live Server에서 변경 사항이 반영되는 것을 확인했습니다.

### 02_my_intro_page

제목, 문단, 목록, 링크를 사용해 간단한 자기소개 페이지를 구성했습니다. `h1` 제목을 수정하고, 목록 항목을 추가하며, 다른 HTML 파일로 이동하는 링크 경로를 직접 수정했습니다.

### 03_shortcut_links

자주 사용하는 사이트로 이동하는 바로가기 링크를 구성했습니다. `a` 태그의 `href`와 `target="_blank"`를 사용해 링크 목적지와 새 탭 열기 동작을 확인했습니다.

## 01_회원가입 & 결과 페이지

HTML Form과 CSS 기초를 종합해 회원가입 페이지와 결과 페이지를 만들었습니다.

회원가입 페이지에서는 `form`, `input`, `select`, `textarea`, `button`을 사용했고, 이름, 아이디, 비밀번호, 이메일, 전화번호, 생년월일, 성별, 관심 분야, 지역, 자기소개, 약관 동의 항목을 구성했습니다. `required`, `pattern`, `minlength`, `maxlength`, `min`, `max`를 적용해 브라우저 기본 유효성 검증도 함께 확인했습니다.

제출 시에는 `result.html`로 이동하도록 만들고, 전달된 입력값을 결과 페이지에서 표로 보여주도록 구현했습니다. 관심 분야처럼 여러 개 선택될 수 있는 값은 복수 값으로 처리했습니다. CSS에서는 `fieldset`, 입력창, 버튼, 테이블의 여백과 테두리를 정리하고, 고정 헤더와 `:focus`, `:hover` 스타일을 적용했습니다.

## 02_여행지 소개 페이지

시맨틱 태그와 미디어 요소를 사용해 여행지 소개 페이지를 만들었습니다.

전체 구조는 `header`, `nav`, `main`, `section`, `article`, `aside`, `footer`로 나누어 작성했습니다. 상단 내비게이션은 `position: fixed`로 고정하고, 각 메뉴를 섹션 `id`와 연결해 페이지 안에서 부드럽게 이동하도록 구성했습니다.

명소 갤러리는 `figure`와 `figcaption`을 사용해 이미지와 설명을 묶었고, 모든 이미지에는 의미 있는 `alt`를 작성했습니다. 먹거리 섹션은 본문 `article`과 팁 `aside`를 2단으로 배치했습니다. 영상 섹션에서는 `video`, `controls`, `poster`, `source`를 사용해 직접 재생할 수 있는 미디어 영역을 구성했습니다.

CSS에서는 히어로 배경에 이미지와 `gradient` 오버레이를 겹치고, `position: absolute`와 `transform`으로 텍스트를 중앙 배치했습니다. 카드 이미지에는 `:hover`와 `scale` 효과를 적용해 마우스를 올렸을 때 확대되는 동작을 확인했습니다.

## 정리

이번 Day1 종합실습을 통해 HTML 문서 구조, 링크, 폼, 유효성 검증, 시맨틱 태그, 이미지와 영상 요소, CSS Box Model, 고정 배치, hover/focus 스타일을 한 흐름으로 연결해볼 수 있었습니다. 기초 태그를 단순히 외우는 것보다, 각 태그가 페이지 안에서 어떤 역할을 하는지 생각하며 작성하는 것이 중요하다는 점을 배웠습니다.
